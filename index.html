Map | Police Shootings

This map depicts incidents of police shooting civilians. Marker color indicates if the victim was only hit (blue), if they were killed (red), or if the report is not specfic (purple). An optional layer highlights those shootings where more than 25 bullets were reported fired (the size of the circle is related to the number of reported shots). Click on a marker for more information including the details of the incident (if available) and a source for the report.

Data is crowd-sourced and provided by Deadspin.
